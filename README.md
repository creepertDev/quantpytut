# quantpytut

Jupyter notebooks covering quantitative finance fundamentals in Python.

## Notebooks

| Notebook | Topic |
|----------|-------|
| `importingData.ipynb` | Pulling market data (yfinance, APIs) |
| `plotingData.ipynb` | Visualising price series and returns |
| `returns.ipynb` | Daily/log returns, cumulative performance |
| `volatility.ipynb` | Realised volatility, rolling std, annualisation |
| `betaWeighting.ipynb` | Portfolio beta weighting against a benchmark |

## Setup

```bash
pip install numpy pandas matplotlib yfinance scipy
jupyter notebook
```
